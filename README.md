# ResQNet

ResQNet is an offline-first emergency communication system designed for disaster situations where internet or telecom networks fail.

## Problem
During floods, earthquakes, and disasters, internet and mobile networks often become unavailable or overloaded. People in the same local area cannot communicate easily.

## Solution
ResQNet allows devices connected to the same local network (Wi-Fi or hotspot) to communicate in real-time without internet.

## Features
- Local server-based communication
- Real-time messaging
- Emergency broadcast alerts
- "I am Safe / Need Help" status
- Local message storage

## Tech Stack
Backend: Node.js, Express, Socket.io  
Frontend: React / HTML  
Database: JSON or SQLite
